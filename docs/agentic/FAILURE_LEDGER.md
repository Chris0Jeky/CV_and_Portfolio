## Entry guidelines

- **Class**: tool, command, file, CDN, browser, git, workflow
- **Surface**: The specific tool or operation that failed
- **Status**: open, resolved, won't-fix
- Only add entries for failures that are **recurring** or **non-obvious**. One-off issues belong in the session response, not here.
